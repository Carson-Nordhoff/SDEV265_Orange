Turn Overview

This page explains what happens on each turn

These are the different phases/steps in a turn:

  1. Beginning Phase (Untap, Upkeep, Draw)
  2. Pre-Combat Main Phase
  3. Combat Phase (Beginning of Combat, Declare Attackers, Declare Blockers, First Strike Damage, Damage, End of Combat)
  4. Post-Combat Main Phase
  5. End Phase (End Step, Cleanup Step)
